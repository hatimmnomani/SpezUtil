# Digital Takeoff UI

Tree-shakeable, low-dependency UI packages for community software. The first component is a
feature-rich **Hijri date picker** where the Hijri (Dawoodi Bohra / Misri) date is primary and the
Gregorian date is secondary.

## Packages

| Package | Description |
| --- | --- |
| `@spezutil/hijri-core` | Zero-dependency Hijri calendar engine. |
| `@spezutil/hijri-datepicker` | `<hijri-datepicker>` Web Component. |
| `@spezutil/hijri-datepicker-react` | React wrapper. |
| `@spezutil/hijri-datepicker-angular` | Angular standalone component. |
| `@spezutil/hijri-calendar` | `<hijri-calendar>` Web Component (month/week/day/agenda + events). |
| `@spezutil/hijri-calendar-react` | React wrapper. |
| `@spezutil/hijri-calendar-angular` | Angular standalone component. |
| `@spezutil/richtext-editor` | `<spez-richtext>` rich-text editor for Arabic / Lisan-ud-Dawat content. |
| `@spezutil/richtext-editor-react` | React wrapper. |
| `@spezutil/richtext-editor-angular` | Angular standalone component. |

## Install

```bash
# Web Components (vanilla)
npm i @spezutil/hijri-datepicker
npm i @spezutil/hijri-calendar
npm i @spezutil/richtext-editor

# React
npm i @spezutil/hijri-datepicker-react react react-dom
npm i @spezutil/hijri-calendar-react react react-dom
npm i @spezutil/richtext-editor-react react react-dom

# Angular
npm i @spezutil/hijri-datepicker-angular
npm i @spezutil/hijri-calendar-angular
npm i @spezutil/richtext-editor-angular
```
